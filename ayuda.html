Ayuda del Plugin Gestión de Rol 'Alumno'

Descripción

Este plugin permite crear y gestionar un rol de usuario con el nombre 'Alumno'. Por defecto, este rol tiene las mismas capacidades que un 'Suscriptor', es decir, únicamente puede leer contenidos. El rol se crea automáticamente al activar el plugin y puede ser eliminado o desactivado desde la administración del mismo.

Cómo gestionar el rol 'Alumno'

Al activarse, el plugin crea automáticamente el rol 'Alumno' con capacidad de lectura.

Una vez activado, puedes:

  • Añadir o eliminar capacidades adicionales utilizando código adicional o un plugin de gestión de roles compatible.
  • Eliminar el rol 'Alumno', si ya no es necesario, a través de las opciones del plugin.

Instalación y Activación

  1. Sube la carpeta del plugin al directorio /wp-content/plugins/ de tu instalación de WordPress.
  2. Activa el plugin a través del menú 'Plugins' en tu panel de control de WordPress.

Desactivación y Eliminación

  1. Para desactivar el plugin, simplemente desactívalo desde la sección 'Plugins' en tu panel de control.
  2. Para eliminar el rol 'Alumno', utiliza la funcionalidad provista dentro de los ajustes del plugin o un plugin de gestión de roles.

Advertencia

Al desactivar o eliminar el plugin, el rol 'Alumno' y cualquier configuración asociada a él podrían ser también eliminados. Asegúrate de comprender las implicaciones de esto y de tener un respaldo de tu configuración si es necesario.
